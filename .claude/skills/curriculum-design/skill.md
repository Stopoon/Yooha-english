# Curriculum Design — 유하 영어 커리큘럼 설계

초등 3학년 수준에 맞는 파닉스·어휘 커리큘럼을 설계하고 `_workspace/01_curriculum_spec.md`에 출력한다.

## 커리큘럼 설계 원칙

8~9세 아이의 학습 특성을 반영한다:
- **반복과 패턴**: 같은 패턴을 다른 단어로 3회 이상 반복 노출
- **누적 학습**: 이전 세션 어휘가 다음 세션에서 자연스럽게 재등장
- **15~20분 완결**: 하루 세션이 이 시간 안에 끝나도록 분량 조절
- **자연스러운 노출**: 문법 규칙 설명 대신 패턴 반복으로 언어 감각 형성

## 파닉스 6레벨 설계

각 레벨은 5개 세션으로 구성. 세션 5마다 동화/만화 복습 콘텐츠 제공.

```
레벨 1 (5세션): 단모음 a, e, i, o, u + 기본 자음 s, t, n, m, p, b
레벨 2 (5세션): 단어 패밀리 -at, -an, -en, -in, -on, -un
레벨 3 (5세션): 나머지 자음 c, d, f, g, h, j, k, l, r, v, w
레벨 4 (5세션): 이중자음 sh, ch, th, wh, ph
레벨 5 (5세션): 장모음 (silent e) — make, like, note, cute
레벨 6 (5세션): 이중모음 ai, ay, ea, ee, oa, ou, ow
```

## 어휘 테마 설계

유하의 관심사와 연결되는 테마를 우선 배치한다.

```
테마 1 — 동물 친구 (레벨 1~2): dog, cat, rabbit, puppy, kitten, bunny, fish, bird
테마 2 — 가족 (레벨 1~2): mom, dad, grandma, grandpa, aunt, uncle, family
테마 3 — 학교와 친구 (레벨 2~3): friend, school, class, desk, book, pencil, play
테마 4 — 자연과 음식 (레벨 3~4): apple, rice, soup, carrot, tree, flower, sun, rain
테마 5 — 감정과 행동 (레벨 4~5): happy, run, jump, sing, draw, read, sleep, eat
테마 6 — 집과 일상 (레벨 5~6): home, room, door, window, bed, table, chair, lamp
```

**금지 어휘**: pizza, hamburger, burger — 발견 즉시 다른 음식 어휘로 교체

## 세션 구조 (30분 → 15~20분으로 압축)

```
파닉스 파트 (5분):
- 오늘의 파닉스 패턴 소개 (동물 캐릭터 뭉치가 소리 설명)
- 소리 따라하기 3~5회 반복

어휘 파트 (5분):
- 오늘의 단어 8~10개 (카드 플래시)
- 단어 + 그림 + 한국어 뜻 병행

게임 파트 (5~7분):
- 4종 게임 중 1개 (세션마다 순환)
- 게임 결과: ★ 1~3개 (점수 압박 없이)

복습 파트 (3분):
- 오늘 배운 단어 5개 빠른 복습
- 맞추면 "잘했어! 뭉치가 기뻐해요 🐶"
```

## 복습 트리거 기준

```
동화/만화 해금 조건:
- 5개 세션 완료마다 동화 1편 해금
- 레벨 완료 시 특별 만화 1편 해금

레벨업 기준:
- 어휘 정답률 80% 이상인 세션 4개 이상 완료 시 다음 레벨로
- 레벨업 시 "뭉치가 커졌어요! 레벨 2로 올라갔어요! 🎉" 메시지
```
