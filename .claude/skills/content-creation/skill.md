# Content Creation — 유하 맞춤 학습 콘텐츠 제작

커리큘럼 스펙을 받아 유하의 세계가 녹아든 구체적 학습 콘텐츠를 `_workspace/02_content_sessions.md`에 생성한다.

## 콘텐츠 개인화 원칙

유하가 "이 프로그램이 나를 알고 있다"는 느낌을 받으려면, 콘텐츠에서 유하의 현실 세계가 보여야 한다.

**모든 예문에 이 요소 중 최소 1개 포함:**
- 친구 이름: 하원(Hawon), 하윤(Hayun), 채율(Chaeyul), 준희(Junhee), 재희(Jaehee), 지민(Jimin)
- 동물 캐릭터: 뭉치(Moongchi, 강아지), 냥이(Nyangi, 고양이), 콩이(Kongi, 토끼)
- 가족: 외할머니(grandma), 엄마(mom), 아빠(dad)

**금지:**
- 피자, 햄버거, burger — 발견 시 즉시 교체 (집밥, 국, 과일, 채소로 대체)
- "틀렸어요", "오답", "점수", "시험" 문구 — 대신 격려 표현 사용

## 파닉스 예문 생성 패턴

각 세션에 유하 친구 또는 가족이 등장하는 예문 5~7개:

```
레벨 1 단모음 'a' 예문 예시:
- "Hawon has a cat." (하원이가 고양이를 가지고 있어요)
- "The cat sat on a mat." (고양이가 매트 위에 앉았어요)
- "Moongchi ran and ran." (뭉치가 달리고 또 달렸어요)
```

예문 작성 규칙:
- 오늘의 파닉스 패턴 단어 2개 이상 포함
- 문장 길이: 4~6 단어
- 한국어 번역 필수 병기

## 어휘 카드 데이터 구조

```json
{
  "word": "rabbit",
  "pronunciation": "래-빗",
  "meaning_kr": "토끼",
  "example_en": "Kongi is a white rabbit.",
  "example_kr": "콩이는 하얀 토끼예요.",
  "character": "kongi",
  "theme": "animals",
  "level": 1
}
```

## 게임 4종 상세 명세

### 게임 1: 소리 맞추기 (Phonics Match)
```
화면: 동물/사물 그림 카드 1장 + 알파벳 보기 4개
동작: 단어의 첫소리 알파벳을 탭
성공: "맞아요! 🐶 뭉치가 신나요!" + 파닉스 소리 재생
실패: "다시 해볼까요? 뭉치가 응원해요!" + 힌트 표시
데이터: { type: "phonics_match", word, correct_letter, options: string[4] }
```

### 게임 2: 단어 조립 (Word Builder)
```
화면: 단어 이미지 + 섞인 글자 타일
동작: 글자 타일을 드래그해서 올바른 순서로 배열
성공: "완성! 냥이가 기뻐해요 🐱" + 단어 발음 재생
실패: "조금만 더! 거의 다 왔어요!" + 첫 글자 힌트
데이터: { type: "word_builder", word, shuffled_letters: string[] }
```

### 게임 3: 카드 뒤집기 (Memory Match)
```
화면: 4×3 그리드 (12장) — 영어 단어 6개 + 한국어 뜻 6개
동작: 카드 2장씩 뒤집어 영어-한국어 짝 맞추기
성공: "짝 찾았어요! 콩이가 폴짝 뛰어요 🐰"
완료: "대단해요! [이름]처럼 잘 기억해요!"
데이터: { type: "memory_match", pairs: [{en, kr}], grid_size: "4x3" }
```

### 게임 4: 문장 완성 (Sentence Fill)
```
화면: 친구 이름이 등장하는 문장 + 빈칸 + 보기 3개
예시: "Hawon loves her ___." 보기: [cat / run / blue]
성공: "정답이에요! 하원이도 기뻐할 거예요!"
데이터: { type: "sentence_fill", sentence_with_blank, answer, options: string[3], friend_name }
```

## 성취 배지 및 격려 문구

```
배지 목록:
- 🌟 첫 세션 완료: "유하의 영어 여행이 시작됐어요!"
- 🔥 3일 연속: "3일 연속 학습! 뭉치가 자랑스러워해요!"
- 🌈 레벨업: "레벨 [N]로 올라갔어요! 정말 대단해요!"
- 💎 만점 세션: "완벽해요! 냥이처럼 똑똑해요!"
- 📚 동화 해금: "새 동화가 열렸어요! 콩이와 함께 읽어봐요!"

격려 문구 모음:
- 정답 시: "맞아요! 👍", "완벽해요! 🌟", "잘했어요!"
- 오답 시: "다시 해볼까요? 🐶", "거의 다 왔어요!", "뭉치가 응원해요!"
- 세션 완료: "오늘도 열심히 했어요! 내일 또 만나요! 🎉"
```
